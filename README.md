# 桃園國際機場空側駕駛許可證測驗 App

這是一個使用 **SwiftUI** 開發的 iOS 原生應用程式，專為準備考取「桃園國際機場空側駕駛許可證」的人員設計。本 App 內建完整的 304 題最新題庫，幫助駕駛員隨時隨地在手機上進行測驗與複習。

## 📱 核心功能 (Features)

* **完整題庫支援**：內建所有交通道規則、安全規範與標誌手勢的 304 題精選考題與圖文。
* **隨機抽題測驗**：每次測驗皆自動從題庫中隨機抽出 20 題，模擬真實考試體驗。
* **選項隨機打亂**：為了達到最佳學習效果，每一題的四個選項順序會自動洗牌，避免使用者死記硬背位置。
* **圖文並茂**：支援針對特殊題型的附圖顯示，幫助理解標誌與現場規範。
* **即時成績結算**：答題完畢後立即計算得分。
* **錯題複習機制**：測驗成績畫面中，會優先把「答錯的題目」置頂顯示，並以紅色字體標示您的錯誤答案與綠色字體標示正確答案，幫助加強記憶。
* **一鍵重新測驗**：成績頁面提供「重新考題」功能，無須退回首頁即可立刻開始下一輪全新挑戰。
* **排版最佳化**：針對中文考題文字進行了防裁切與換行處理，關閉系統強制對稱功能，確保長篇題目與選項完整呈現於任何尺寸的 iPhone 螢幕。

## 🛠 開發技術 (Tech Stack)

* **語言**: Swift 5+
* **UI 框架**: SwiftUI
* **架構**: MVVM (Model-View-ViewModel)
    * `Question`: Data Model (資料模型)
    * `QuizViewModel`: 負責測驗狀態管理、計分、隨機抽取、打亂邏輯與錯題排序。
    * `ContentView`: 主視圖 UI 渲染。

## 📂 專案結構 (Project Structure)

* `AirportQuizApp.swift`: App 切入點 (Entry point)
* `ContentView.swift`: 包含首頁 (`StartView`)、測驗單元 (`QuizView`) 以及結算與複習畫面 (`ResultsView`) 的所有 UI 元件與視圖模型。
* `Questions.swift`: 輕量化的本地端題庫，包含所有的 Swift Struct `Question` 陣列定義。
* `Assets.xcassets`: 存放首頁 Icon 及題庫附帶之影像檔案資源 (qX_imageXX)。

## 🚀 執行與測試 (How to Run)

1. 確認您已安裝 **Xcode 15** 或更新版本。
2. 點擊 `機場駕照.xcodeproj` 開啟專案。
3. 在上方裝置選單中選擇您的 iPhone 或任意 iOS 模擬器。
4. 按下 `Cmd + R` 編譯並執行 App。
